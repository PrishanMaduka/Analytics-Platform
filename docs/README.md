## Overview

MxL SDK provides comprehensive mobile app monitoring including:

- **Crash Reporting**: Automatic crash detection and reporting
- **Performance Monitoring**: CPU, memory, startup time, ANR detection
- **Network Monitoring**: Request/response tracking and analysis
- **User Analytics**: Screen views, interactions, user journeys
- **Security**: PII redaction, encryption, GDPR compliance
- **Offline Support**: Local storage and batch upload
- **Real-time Dashboards**: Comprehensive monitoring UI

## Platform Support

- **Android**: API 21+ (Android 5.0+)
- **iOS**: iOS 12.0+
